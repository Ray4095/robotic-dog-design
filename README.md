# robotic-dog-design

This repository contains the mechanical design report for a preliminary robotic dog model.

## Project Description

The project focuses on the initial mechanical design of a simple robotic dog, including:

- Body and frame design
- Leg design
- Joints and degrees of freedom
- Motor selection
- Preliminary torque calculation
- Stability and center of gravity
- Proposed walking gait
- Expected mechanical problems
